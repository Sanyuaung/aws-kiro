# User Personas - Kiro Dash

## Overview
This document defines the user personas for Kiro Dash, representing the target players who will interact with the game.

---

## Primary Persona: The Casual Gamer

### Profile
**Name**: Alex (Casual Player)  
**Player Type**: Casual Gamer  
**Experience Level**: Beginner to Intermediate

### Characteristics
- Plays games for short entertainment sessions (5-15 minutes)
- Prefers simple, intuitive controls
- Enjoys quick-to-learn, hard-to-master gameplay
- Plays during breaks or commute
- Values immediate feedback and clear visual/audio cues

### Goals
- Have fun and relax during short breaks
- Experience a sense of achievement through high scores
- Improve skills gradually over multiple sessions
- Enjoy satisfying game mechanics (jumping, collecting, power-ups)

### Motivations
- Quick entertainment without complex learning curve
- Compete with personal best scores
- Experience progression through increasing difficulty
- Enjoy vibrant visuals and responsive controls

### Frustrations
- Overly complex controls or mechanics
- Unclear game rules or objectives
- Unfair difficulty spikes
- Lack of feedback on actions
- Poor performance or laggy controls

### Behaviors
- Plays multiple short sessions rather than long sessions
- Focuses on beating personal high score
- Appreciates visual and audio feedback
- Expects game to be playable immediately without tutorials
- May play with sound off in public settings

### Technical Context
- Plays on desktop or laptop browser
- Uses keyboard for controls
- Expects smooth 60 FPS performance
- May have varying screen sizes

## Secondary Persona: The Score Chaser

### Profile
**Name**: Jordan (Competitive Player)  
**Player Type**: Achievement-Oriented Gamer  
**Experience Level**: Intermediate to Advanced

### Characteristics
- Plays to achieve high scores and master mechanics
- Willing to replay many times to improve
- Analyzes game patterns and optimal strategies
- Values precise controls and fair difficulty
- Enjoys risk-reward decisions (e.g., collecting coins vs. playing safe)

### Goals
- Achieve the highest possible score
- Master all game mechanics (timing jumps, power-up usage)
- Discover optimal strategies for score maximization
- Experience the full range of game difficulty

### Motivations
- Personal achievement and skill mastery
- Competing against high score records
- Perfecting timing and execution
- Exploring all game features and power-ups

### Frustrations
- Inconsistent collision detection
- Random difficulty that feels unfair
- Lack of skill-based progression
- Controls that don't respond precisely
- Missing high score tracking

### Behaviors
- Plays longer sessions to achieve flow state
- Focuses intensely on gameplay
- Experiments with different strategies
- Pays attention to score multipliers and power-ups
- Expects consistent, predictable game mechanics

### Technical Context
- Plays on desktop with keyboard
- Expects precise, low-latency controls
- Values consistent frame rate
- Appreciates responsive game feel

---

## Persona Usage in Stories

User stories will reference these personas to ensure features serve player needs:
- **Alex (Casual Gamer)**: Stories focused on accessibility, clarity, and immediate fun
- **Jordan (Score Chaser)**: Stories focused on skill mastery, precision, and achievement

Both personas benefit from:
- Responsive controls
- Clear visual feedback
- Progressive difficulty
- High score tracking
- Smooth performance
